# Gehirnsegmentierung-Abschlussprojekt-VmedS
Abschlussprojekt für den Kurs "Verarbeitung medizinischer Signale," BME3. In diesem Projekt wird ein Python Programm zur Segmentierung von Gehirn-MRT-Scans in WM, GM und CSF. Dafür brauchen wir von Vorverarbeitung der Daten über die eigentliche Segmentierung bis hin zur Visualisierung der Ergebnisse.
